BabySpace — 신생아 육아 기록 앱

시행일: 2026년 4월 21일

1개요

베이비스페이스(이하 "앱")는 신생아·영유아를 돌보는 부모를 위한 육아 기록 서비스입니다. 개발자(이하 "운영자")는 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령에 따라 이용자의 개인정보를 보호하고 있습니다.

본 방침은 앱이 수집하는 정보의 종류, 수집 목적, 보유 기간, 제3자 제공 여부 등을 안내합니다.

앱은 아래 정보를 수집하며, 수집된 정보는 앱 기능 제공 외의 목적으로 사용하지 않습니다.

4개인정보 제3자 제공

운영자는 이용자의 개인정보를 원칙적으로 제3자에게 제공하지 않습니다. 다만, 아래의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우 (가족 공유 기능 사용 시 Firebase를 통해 가족 구성원과 공유)
  • 법령에 의하여 제공이 요구되는 경우

5외부 서비스 연동

앱은 아래 외부 서비스를 사용합니다. 각 서비스의 개인정보처리방침을 함께 확인해주세요.

서비스 제공사 목적

Google의 개인정보처리방침: https://policies.google.com/privacy

6개인정보 보유 및 이용 기간

  • 로컬 저장 데이터: 앱을 삭제하거나 이용자가 직접 삭제할 때까지 기기에 보관됩니다.
  • Firebase 동기화 데이터: 가족 공유 기능 해제 또는 앱 내에서 데이터를 삭제하면 서버에서도 삭제됩니다.
  • 법령에서 별도 보유 기간을 정한 경우 해당 기간을 따릅니다.

7개인정보 삭제 방법

  • 기기 내 데이터: 앱 삭제 시 모든 로컬 데이터가 자동으로 삭제됩니다.
  • 기록 단건 삭제: 앱 내 기록 목록 또는 캘린더에서 삭제 버튼을 통해 즉시 삭제 가능합니다.
  • Firebase 데이터 삭제 요청: 아래 이메일로 요청하시면 처리해드립니다.

8아동 개인정보 보호

본 앱의 사용자는 아기를 돌보는 부모 및 보호자(만 18세 이상)입니다. 앱에서 기록되는 아기 정보(이름, 생년월일 등)는 보호자가 직접 입력·관리하며, 아동이 직접 앱을 이용하는 서비스가 아닙니다.

수집된 아기 정보는 육아 기록 기능 제공 목적으로만 사용되며, 마케팅·광고 타겟팅에 사용되지 않습니다.

9광고 및 추적

앱은 Google AdMob을 통해 광고를 제공합니다. AdMob은 광고 효과 측정을 위해 광고 ID 등 기기 정보를 수집할 수 있습니다.

  • 광고 개인화를 원하지 않으시면 기기 설정 → 개인정보 보호 → 광고에서 '광고 맞춤설정 선택 해제'를 이용하실 수 있습니다.
  • 광고 수익은 앱의 무료 운영에 사용됩니다.

10이용자의 권리

이용자는 아래 권리를 언제든지 행사할 수 있습니다.

  • 수집된 개인정보의 열람 요청
  • 수집된 개인정보의 정정·삭제 요청
  • 개인정보 처리 정지 요청
  • 개인정보 이동 요청

권리 행사는 아래 개인정보 보호책임자에게 이메일로 요청하실 수 있으며, 확인 후 지체 없이 처리하겠습니다.
